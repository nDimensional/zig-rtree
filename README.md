Generic RTree for Zig vectors

## Table of Contents

- [License](#license)

## Installation

This library uses and requires Zig version `0.14.0` or later.

Add the dependency to `build.zig.zon`:

```zig
.{
    .dependencies = .{
        .sqlite = .{
            .url = "https://github.com/nDimensional/zig-rtree/archive/refs/tags/v0.2.0.tar.gz",
            // .hash = "...",
        },
    },
}
```
